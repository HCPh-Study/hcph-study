# hcph-study

The YODA super-repo for the Human Connectome PHantom (HCPh) study.

## Dataset structure

- All inputs (i.e. building blocks from other sources) are located in `inputs/`.
  - `inputs/data/` — the BIDS raw dataset
- All custom code is located in `code/`.
- Processing derivatives will be added as subdatasets under `derivatives/`.
